# Project Concept

## Collaborative Dashboard Builder

A polished web app where users create analytics dashboards by adding widgets, arranging them on a grid, applying filters, and seeing lightweight realtime presence and live cursors from other viewers/editors.

## Product Feel

### Should feel like

- modern internal analytics software
- frontend-heavy and interaction-rich
- polished and demo-ready
- real enough backend to feel credible

### Should not feel like

- a backend platform
- a BI engine
- a CRDT research project
- a visual-only prototype without architecture

## Why This Project Exists

Existing portfolio projects already show:

- backend and system design depth
- infrastructure and product architecture
- full-stack business logic

This project adds:

- rich client-side state architecture
- advanced React UI composition
- interaction-heavy UX
- visual polish
- realtime features without backend bloat

## Core Product Pitch

Users can:

- create a dashboard
- add chart widgets
- drag and resize widgets
- apply global filters
- edit widget settings
- save and revisit dashboards
- share a URL
- open in multiple tabs and see connected users, remote cursors, and selected widget presence
